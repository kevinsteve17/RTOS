Procedimiento
-------------
1. Extraer folder

2. El folder comprimido contiene los siguientes folder:
    - Proyecto2
    - Documentation

3. Para compilar el proyecto, se debe de ejecutar el comando 'make' dentro del folder "Proyecto2". Despues de compilar y vincular el programa se crearan 2 folders:
    - Proyecto2/build
    - Proyecto2/dist/Debug/GNU-Linux

4. El ejecutable se encontrara en el folder:
    - Proyecto2/dist/Debug/GNU-Linux/

5. Para ejecutar el binario se usa el comando 
    './proyecto2'

6. Todos los archivos de salida incluyendo el codigo de latex quedara en este folder despues de una calendarizacion
